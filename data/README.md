# Data Overview

- 이 폴더는 **SceneClick** 프로젝트에서 사용하는 데이터셋을 저장하는 공간입니다.
- 주요 데이터는 유튜브 해외여행 썸네일의 이미지 정보와 사용자 반응(조회수, 구독자 수 등) 데이터를 포함하고 있습니다.


## 파일 목록

 - **thumbnail_data.xlsx**: 썸네일 분석을 위한 원본 데이터셋입니다.


## 데이터 구조 (Data Dictionary)

`thumbnail_data.xlsx` 파일은 다음과 같은 컬럼으로 구성되어 있습니다.

| 컬럼명 (Column) | 설명 (Description) | 데이터 예시 |
| :--- | :--- | :--- |
| **밝기(0~255)** | 썸네일 이미지 밝기 | `124.5` |
| **대비(0~255)** | 썸네일 이미지 대비 | `58.2` |
| **채도(0~1.0)** | 썸네일 이미지 채도 | `0.48` |
| **배경 대비 얼굴 면적 비율(%)** | 썸네일 크기 대비 얼굴(몸 제외) 면적 비율(%) | `3.8` |
| **음절 수** | 썸네일 제목 음절 수 | `8` |
| **텍스트 면적 비율(%)** | 썸네일 크기 대비 제목 면적 비율(%) | `8.5` |
| **특수부호개수** | 썸네일 제목 특수부호 개수 ex) ',','!','()' | `2` |

| **최신성** | 2024년 해외여행 브이로그 한국 유튜버 대상 (13 - 업로드 게시월) | `10` |
| **채널 영상 총 조회수** | 2026년 1월 6일 화요일 정보 수집 당시 기준 | `125903890` |
| **조회수** | 2026년 1월 6일 화요일 정보 수집 당시 기준 | `431627` |
